Lawrence's Portfolio

This repository contains a small static portfolio website for Lawrence. Use it to showcase projects, skills, and contact information.

## Project files

- [index.html](index.html) — main portfolio page
- [preview.html](preview.html) — preview / alternate page
- [script.js](script.js) — site JavaScript
- [style.css](style.css) — site styles

## Overview

This is a lightweight, static portfolio intended as a starting point you can customize and deploy to GitHub Pages, Netlify, or any static host.

## Suggested sections to add

- Projects (screenshots, descriptions, links)
- About / bio
- Contact (mailto link or contact form)
- Resume / downloadable PDF

## Customize

1. Edit [index.html](index.html) to update content and project entries.
2. Change appearance in [style.css](style.css).
3. Add interactive behavior in [script.js](script.js).

## Local preview

Open [index.html](index.html) in your browser, or run a simple server:

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000/
```

## Deployment

- GitHub Pages: push the repo and enable Pages in settings
- Netlify / Vercel: connect the repo for continuous deployment
